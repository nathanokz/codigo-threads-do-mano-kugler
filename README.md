Código referente ao exercício:

Implemente um programa multithread que:

- Aloca um vetor double com 200 milhões de entradas

- Inicializa o vetor alocado anteriormente com valores aleatórios

- Após a inicialização exibe uma mensagem ‘Encerrou inicalizacao’

- Computa e exibe a quantidade de valores no vetor que são maiores que 0.25 e menores que 0.75
